# Shapes and Lasers

Shapes and Lasers is a 2D arcade shooter built with the Godot Engine. Inspired by classic arcade games such as *Galaga* and modern bullet hell titles like *Touhou Project*, the game focuses on fast-paced gameplay, precise movement, and surviving increasingly difficult enemy waves.

The game embraces a minimalist visual style, using simple geometric shapes for ships, enemies, and projectiles while delivering challenging and satisfying gameplay.

## Features

* Fast-paced arcade shooter gameplay
* Bullet hell inspired enemy patterns
* Minimalist graphics using geometric shapes
* Increasing difficulty through enemy waves
* Responsive player movement and shooting
* Built with the Godot Engine

## Gameplay

Pilot your ship through waves of enemies while avoiding dense bullet patterns. Success depends on precise movement, quick reactions, and learning enemy attack behaviors.

As the game progresses, enemies become faster, more aggressive, and introduce increasingly complex projectile patterns inspired by classic shoot 'em up games.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/shapes-and-lasers.git
cd shapes-and-lasers
```

Open the project using the Godot Engine and press **Play** to run the game.

## Controls

| Key               | Action       |
| ----------------- | ------------ |
| Arrow Keys / WASD | Move         |
| Space             | Focus (Slow Movement)        |
| Escape            | Pause / Quit |

*Controls may change during development.*

## Design Goals

The project was created to explore the design principles behind classic arcade shooters while maintaining a clean, minimalist presentation. By stripping visuals down to basic shapes, the gameplay becomes the primary focus, emphasizing movement, positioning, and player skill.
